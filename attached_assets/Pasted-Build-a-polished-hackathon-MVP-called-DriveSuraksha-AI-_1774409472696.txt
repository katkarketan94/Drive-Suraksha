Build a polished hackathon MVP called “DriveSuraksha AI” — an AI-powered driver safety and civic intelligence platform for Indian urban roads.

Context:
This prototype is for a hackathon. The goal is not full production accuracy; the goal is a convincing, demo-ready working product that shows the workflow end-to-end and matches the concept of:

- traffic rule violation detection
- two-wheeler risk prediction
- lane discipline monitoring
- honking / civic behavior scoring
- accident hotspot prediction
- real-time risk alerts

- performance insights dashboard

Build for rapid prototyping in 2 days. Prioritize a clean UI, strong storytelling, and a complete demo flow over perfect ML accuracy.

IMPORTANT PRODUCT DIRECTION:

1) a driver-facing mobile-first experience

3) a mock real-time AI inference pipeline
4) a civic scoring system
5) a route risk / hotspot view
6) a demo mode using sample video/image/GPS events

Use mocked or simulated data where needed, but make the app look and behave like a real AI product.

TECH STACK:
Use a modern Replit-friendly stack:
- Frontend: Next.js + React + Tailwind CSS

  - smooth driving

5) City Intelligence Dashboard

6) Prototype / Demo Mode

  Step 3: lane violation risk appears

  Step 5: civic score changes
  Step 6: hotspot recommendation appears
- The demo should automatically advance through these events with smooth transitions
- Include a visible event log panel with timestamps and AI decisions

CORE FUNCTIONAL LOGIC:
Implement a simple rules-based scoring engine so the app feels intelligent even without real ML.

Example scoring logic:
- Start civic score at 85
- -15 for pothole/waterlogging hazard if repeated
- -10 for lane violation
- -8 for wrong-side driving
- -5 for honking spike
- -12 for helmet missing in two-wheeler mode
- +3 for smooth driving / no incidents over time
- risk level:
  - LOW if score >= 75
  - MODERATE if score between 50 and 74
  - HIGH if score < 50

Create a function that:
- ingests simulated events
- updates civic score
- updates risk level
- generates a recommended action
- appends a human-readable event log entry

Make the logic deterministic and easy to explain in the pitch.
